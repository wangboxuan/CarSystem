5.   `(2)在集合下创建表users`
6.   `(3)在users里添加数据 `

8.   `"yonghuming": "kai",`
9.   `"mima": "14bf051807d009959b18677b216d0bea8144a4ae37da9d23e6aee73189c60e95",`
10.  `"__v": 0.0`
11.  `})`
12.   `复制这段代码，因为用户登录界面密码是需要加密的 `
   
13. 初始化`cnpm i --save`
14. 运行 `app.js`
